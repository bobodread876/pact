# @pact/pactd *(stub)*

The sidecar daemon — the **universal adapter**. A local long-running process that holds keys (isolated from the agent) and exposes bond operations over `127.0.0.1` (HTTP/JSON + SSE), so the relationship layer runs *alongside* any agent platform in any language. Will also expose the sats/Lightning payment interface for the layered markets ([ECONOMICS.md](../../ECONOMICS.md)).
